## STACK
React Native + Expo (Front) + Spring Boot (Backend)

## Ejecutar Back
***Crear modificar .env.template y renombrarlo a .env***
```
cd backend/loocks
```
```
./gradlew bootRun
```

## Ejecutar Front
```
cd mobile
```
```
npm start
```

## Cargar dependencias en Front
```
npm i
```

## Ver documentación api
```
http://server:port/swagger-ui/index.html
```
